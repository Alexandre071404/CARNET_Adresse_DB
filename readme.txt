Explication de l'arborescence du projet:

Nous retrouvons 3 répertoires principaux que sont "Bases_de_donnees","Formulaire","PHP".

Le répertoire "Bases_de_donnees" contient le fichier Acteur.sql qui permet de réinitialiser la table "Acteur" de ma base de données. 

Le répertoire "Formulaire" contient le fichiers html qui permet de générer un formulaire fonctionnel utilisé par le site.

Le répertoire "PHP" quant à lui contient toutes les fonctions PHP nécéssaires pour le bon déroulement de l'exécution du site.
Notons également que dans ce répertoire nous retrouvons un répertoire "Protection" dans lequel nous retrouvons le fichier contenant le mot de passe de ma base de donnée.
Ceci a pour but d'éviter d'exposer directement mon mot de passe dans le programme principal de mon site internet.

Le fichier index.php est le fichier qui lance le site et qui puisera dans les diverses ressources des répertoires précédemment cités.



Project Structure Explanation:

There are three main directories: "Databases," "Form," and "PHP."

The "Databases" directory contains the Acteur.sql file, which is used to reset the "Acteur" table in my database.

The "Form" directory contains the HTML files that generate a functional form used by the website.

The "PHP" directory contains all the necessary PHP functions for the proper execution of the site. Additionally, within this directory, there is a "Protection" folder where the file containing my database password is stored. This is to prevent the password from being directly exposed in the main program of my website.

The index.php file is the one that launches the site and pulls from the various resources in the directories mentioned above.
